        kosli version

        kosli version #

        Print the client version information

        Synopsis #

        Print the version for Kosli CLI.

        The output will look something like this: version.BuildInfo{Version:"v0.0.1", GitCommit:"fe51cd1e31e6a202cba7dead9552a6d418ded79a", GitTreeState:"clean", GoVersion:"go1.16.3"}

        • Version is the semantic version of the release.
        • GitCommit is the SHA for the commit that this version was built from.
        • GitTreeState is "clean" if there are no local code changes when this binary was built, and "dirty" if the binary was built from locally modified code.
        • GoVersion is the version of Go that was used to compile Kosli CLI.
        kosli version [flags]
