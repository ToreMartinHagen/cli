    merkely fingerprint #

    Print the SHA256 fingerprint of an artifact.

    Synopsis #

    Print the SHA256 fingerprint of an artifact. Requires artifact type flag to be set. Artifact type can be one of: “file” for files, “dir” for directories, “docker” for docker images.

    merkely fingerprint [flags]

    Options # 

      -h, --help          help for fingerprint
  -t, --type string   The type of the artifact to calculate its SHA256 fingerprint.
